## 🎨 Customization Guide

### 1. Personal Information
Edit the following in `index.html`:
- Name and title in the hero section
- Contact information (email, phone, location)
- Social media links
- Profile image (replace the placeholder URL)

### 2. Skills Section
Update your skills in the skills section:
- Modify skill names and percentages
- Add or remove skill categories
- Update the progress bars

### 3. Projects
Add your projects in the projects section:
- Project images (replace placeholder URLs)
- Project descriptions
- Technology tags
- GitHub and live demo links

### 4. Education & Certifications
Update your educational background:
- Institution names and dates
- Grades and achievements
- Certification details

### 5. Styling
Customize the appearance in `styles.css`:
- Color scheme (primary colors, gradients)
- Fonts and typography
- Animations and transitions
- Layout spacing

## 🚀 Deployment

### Option 1: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload your files
3. Go to Settings > Pages
4. Select source branch (usually main)
5. Your site will be available at `https://username.github.io/repository-name`

### Option 2: Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your project folder
3. Get instant deployment with custom domain option

### Option 3: Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Connect your GitHub repository
3. Automatic deployment on every push
